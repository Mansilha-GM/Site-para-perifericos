Periféricos Store

Uma página web simples e moderna para exibição de periféricos gamers, desenvolvida com HTML, CSS e JavaScript puro.

Sobre o projeto

Este projeto é uma vitrine de produtos organizada por categorias, com foco em:

Interface moderna (tema escuro)
Navegação simples e intuitiva
Organização por categorias
Interação rápida e leve (sem frameworks)
Funcionalidades
Scroll suave até a seção de produtos
Filtro por categorias:
Mouse
Teclado
Headset
Cards de produtos com:
Imagem
Nome
Preço
Clique no produto abre a página de compra
Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla)
